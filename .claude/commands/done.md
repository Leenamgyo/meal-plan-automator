GitHub 이슈를 닫고 TODO.md에서 해당 항목을 완료로 이동한다.

이슈 번호: $ARGUMENTS

## 진행 순서

1. `$ARGUMENTS`에서 이슈 번호를 파악한다. 없으면 사용자에게 번호를 물어본다.

2. `gh issue view [번호]`로 이슈 제목과 상태를 확인한다.
   - 이미 닫혀있으면 TODO.md 업데이트만 진행한다.

3. 이슈가 열려있으면 `gh issue close [번호] --comment "작업 완료"`로 닫는다.

4. `TODO.md`를 읽고 `#[번호]`가 포함된 항목을 찾는다.
   - `## 📋 예정` 섹션에서 해당 항목의 `- [ ]`를 `- [X]`로 변경한다.
   - 항목을 찾지 못하면 사용자에게 알린다.

5. 결과를 사용자에게 보여준다:
   - 닫은 이슈 제목과 번호
   - TODO.md에서 완료 처리된 항목
